# Clone da Netflix

Projeto desenvolvido durante a Imersão Front-End da Alura com o objetivo de recriar a interface da Netflix, incluindo interações com JavaScript.

## Acesse o projeto

👉 https://pedrocassol.github.io/imersao-alura-frontend/

## Sobre o projeto

Este projeto consiste em um clone da interface da Netflix, com foco na seleção de perfis e interações dinâmicas utilizando JavaScript.

Parte do desenvolvimento inicial foi realizado com auxílio de Inteligência Artificial, como forma de aprendizado e exploração de novas ferramentas, com ajustes e melhorias feitos manualmente ao longo do projeto.

## O que eu pratiquei

* Estruturação de páginas com HTML
* Estilização com CSS
* Manipulação do DOM com JavaScript
* Criação de interações na interface
* Uso de IA como ferramenta de apoio no desenvolvimento
* Organização de arquivos

## Funcionalidades

* Alternância entre modo claro e escuro
* Seleção de perfis com armazenamento no navegador
* Navegação entre páginas
* Interface inspirada na Netflix

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
